# Images Folder

## Required Assets

### Hero Video
- **Location**: Add video file here or use external URL
- **Format**: MP4 recommended
- **Size**: Keep under 5MB for fast loading
- **Resolution**: 1920x1080 or higher
- **Source**: Download from [Pexels](https://www.pexels.com)

**To add video:**
1. Download video from Pexels (search for "DJ", "vinyl", "turntable", "record")
2. Place in this folder or use CDN URL
3. Update `index.html` line 30 with video path:
   ```html
   <source src="images/hero-video.mp4" type="video/mp4">
   ```

### Favicon
- **File**: `favicon.ico`
- **Size**: 32x32px or 16x16px
- **Format**: ICO or PNG
- **Design**: Simple "K" in yellow (#FFD53D) on black background

**Generate favicon:**
- Use [RealFaviconGenerator](https://realfavicongenerator.net)
- Or create simple PNG and convert to ICO

### App Screenshots (Optional - for future)
- `hero-app.png` - Main app screenshot for hero section
- `feature-*.png` - Feature screenshots
- Recommended: 1200x800px, optimized for web
