# Screenshots Folder

## Required Screenshots

Add these three app screenshots to this folder:

### 1. collection-view.png
- **Size**: 360x780px (or 720x1560px for retina)
- **Content**: Collection grid view showing organized albums
- **Format**: PNG
- **Optimize**: Keep under 200KB

### 2. playlist-view.png
- **Size**: 360x780px (or 720x1560px for retina)
- **Content**: Playlist view with tracks, drag handles visible
- **Format**: PNG
- **Optimize**: Keep under 200KB

### 3. live-mode.png
- **Size**: 360x780px (or 720x1560px for retina)
- **Content**: Live mode showing NOW PLAYING and UP NEXT cards
- **Format**: PNG
- **Optimize**: Keep under 200KB

## Screenshot Tips

- Export at 2x resolution for retina displays (720x1560px actual)
- Use dark backgrounds to match the design system
- Crop to show clean UI without device chrome
- Optimize with ImageOptim or TinyPNG
- Ensure yellow accents (#FFD53D) are visible

## Temporary Placeholder

If screenshots aren't ready yet, the images will show broken image icons. You can temporarily replace the `<img>` tags with placeholder divs in `index.html` if needed.
