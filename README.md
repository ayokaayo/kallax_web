# Kallax Landing Page

Single-page static website for Kallax, a vinyl DJ companion app.

## Quick Start

1. Open `index.html` in a browser
2. Or use a local server: `python3 -m http.server 8000`

## Setup

- **Form**: Replace `YOUR_FORM_ID` in `index.html` with your Formspree endpoint
- **Video**: Hero video is set to `images/record1.mp4`
- **Screenshots**: Add app screenshots to `images/screenshots/`

## Deployment

Deploy to Vercel, Netlify, or GitHub Pages. See `SETUP.md` for detailed instructions.

## Design System

Neo-brutalist dark theme. See `styleguide/` folder for design system documentation.
